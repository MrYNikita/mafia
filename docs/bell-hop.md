# Bellhop
Данный скрипт позволяет создавать коридорные элементы, между элементами которых можно переключаться, подобно меню в различных приложениях. Это позволяет создавать сложные одностраничные приложения без необходимости реализовывать многостраничники или самописные аналоги.

# HTMLBellhopElement
Основой скрипта являюется кастомный `bell-hop` тег, который создан на основе класса `HTMLBellhopElement`. Данный элемент отвечает за выполнение заложенной в него логики отображения вложенных в него элементов за счёт прописанного внутри `shadowRoot` с определёнными для дочерних элементов стилями.

Пример создания коридорного элемента:

```html
<bell-hop></bell-hop>
```

Для создания коридора, необходимо определить коридорный элемент и вложенные в него конечные точки коридора. Такие точки обозначаются тегом `bell-point` и позволяют понять коридору, какие у коридора существуют энд-поинты, в которые может прийти пользователь.

Коридор понимает какой именно `bell-point` нужно отобразить за счёт атрибута `active`. Данный атрибут указывает на то, что сейчас пользователь находится в этом энд-поинте. Все остальные энд-поинты являются неактивными и их контент не отображается. Активный энд-поинт занимает всё доступное ему пространство родительского коридора.

Также коридор берёт начало от энд-поинта с меткой `root`. Указание таких энд-поинтов обязательно, т.к. с их помощью коридоры понимают, откуда следует начинать прокладывать сложные маршруты.

Пример создания коридора с эндпоинтами:

```html
<bell-hop>
    <bell-point name="A" root active></bell-point>
    <bell-point name="B"></bell-point>
    <bell-point name="C"></bell-point>
</bell-hop>
```

Коридоры также работают с вложенными в друг друга эндпоинтами. В случае, если есть активный энд-поинт вложенный в другой энд-поинт, то:

- Энд-поинт родитель будет отображаться, но вложенные в него элементы, кроме активного вложенного энд-поинта, отображаться не будут;
- Активный энд-поинт будет отображён;

Пример глубоко вложенного энд-поинта:

```html
<bell-hop>
    <bell-point name="A" root></bell-point>
    <bell-point name="B"></bell-point>
    <bell-point name="C">
        <bell-point name="C2" active></bell-point>
    </bell-point>
</bell-hop>
```

До данного примера были показаны отображения энд-поинтов, но не были продемонстрированы способы перемещения между ними. Для осуществления подобного перехода следует использовать `bell-button`. Данный элемент должен иметь атрибут `next`, который должен быть равен наименованию `bell-point`, до которого он должен добраться.

Пример с реализацией переходов:

```html
<bell-hop>
    <bell-point name="home">
        <bell-button next="A">A</bell-button>
        <bell-button next="B">B</bell-button>
        <bell-button next="C">C</bell-button>
    </bell-point>
    <bell-point name="A" root active>
        <bell-button next="home">Home</bell-button>
    </bell-point>
    <bell-point name="B">
        <bell-button next="home">Home</bell-button>
    </bell-point>
    <bell-point name="C">
        <bell-button next="home">Home</bell-button>
    </bell-point>
</bell-hop>
```

С помощью классов `HTMLBellhopElement` и `HTMLBellhopPoint` можно осущесвлять различные манипуляции над энд-поинтами, например, управляя переключениями через скрипты или назначая на переходы обработчики.
